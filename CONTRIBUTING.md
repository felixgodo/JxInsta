## 💡 I Want To Contribute

> When contributing, make sure all work is your own and you have rights to share it. All contributions will be provided under the project’s license.

### 🐞 Reporting Bugs

Before submitting a bug:
- Ensure you're on the latest version
- Confirm it's a bug (and not a misconfiguration)
- Search the [issue tracker](https://github.com/ErrorxCode/JxInsta/issues) to avoid duplicates
- Gather info:
  - Stack trace or logs
  - OS and environment details
  - Reproduction steps

Submit a bug report by:
- Opening a [new issue](https://github.com/ErrorxCode/JxInsta/issues/new)
- Providing expected vs. actual behavior
- Sharing a minimal reproducible example if possible

> 🔐 Please do **not** post security issues publicly. Email them privately if needed.

### 🚀 Suggesting Enhancements

Before submitting a feature suggestion:
- Verify the feature isn’t already implemented or suggested
- Make sure it's in scope with the project’s goals
- Think about broader impact and usability

To suggest an enhancement:
- Open a [GitHub issue](https://github.com/ErrorxCode/JxInsta/issues)
- Use a clear title and detailed description
- Explain current limitations and your proposed improvement
- Optional: Include visuals or references to similar tools

## 🧪 Contribution Guide

To contribute:
- Knowledge of Java and OOP is recommended
- Familiarity with **OkHttp** is a plus

The project relies on an OpenAPI specification that describes Instagram's private endpoints. You may:
- Help discover undocumented endpoints (reverse engineering experience helps)
- Wait until the OpenAPI spec is published and start implementing defined requests

> To contribute, simply fork the repository, create a branch, and open a pull request when ready.

---
## 📌 Summary

You can contribute in two main ways:

- 🔍 By reverse-engineering and documenting new Instagram private API endpoints
- 💻 By implementing already documented endpoints into the Java library

For any support or feedback, please **open an issue** on GitHub — we’ll respond as soon as possible.

Thank you for helping make **JxInsta** better! 🙏
